当前设备未启用 IPv6 网络

本服务仅支持 IPv6 访问，请先启用 IPv6 或通过下方按钮检测网络状态，按指南配置后重试～

简单 IPv6 启用方法
🖥️ 电脑（Windows 10/11）
  • 1. 右键点击桌面「网络」图标 → 选择「网络和 Internet 设置」
  • 2. 找到当前连接的网络（Wi-Fi/以太网）→ 点击进入「属性」
  • 3. 勾选「Internet 协议版本 6 (TCP/IPv6)」→ 点击「确定」
  • 4. 重启网络连接（断开重连 Wi-Fi/网线）即可生效
💻 电脑（macOS）
  • 1. 打开「系统设置」→ 点击「网络」
  • 2. 选择当前连接的网络 → 点击「高级」→ 切换到「TCP/IP」标签
  • 3. IPv6 配置选择「自动」→ 点击「好」保存
  • 4. 断开重连网络即可启用
📱 手机（iOS）
  • 1. 打开「设置」→ 「无线局域网」→ 点击当前 Wi-Fi 后的「i」
  • 2. 找到「IPv6」→ 选择「自动」
  • 3. 关闭 Wi-Fi 重新连接，即可启用
📱 手机（Android）
  • 1. 打开「设置」→ 「WLAN」→ 长按当前连接的 Wi-Fi → 选择「修改网络」
  • 2. 勾选「显示高级选项」→ IPv6 协议选择「自动获取」
  • 3. 点击「保存」，重启 Wi-Fi 即可生效（不同品牌路径略有差异，找不到可搜索手机型号+IPv6 启用）
📡 路由器（常见品牌通用步骤）
  • 1. 电脑/手机连接路由器 Wi-Fi，打开浏览器输入路由器管理地址（常见：192.168.1.1、192.168.0.1，路由器底部标签可查）
  • 2. 输入管理员账号密码（默认 admin/admin，未修改过可查说明书或路由器底部）
  • 3. 找到「网络设置」→ 「IPv6 设置」（部分品牌在「高级设置」或「上网设置」中）
  • 4. 选择 IPv6 模式为「自动获取（SLAAC/DHCPv6）」或「原生 IPv6」，启用后点击「保存」
  • 5. 重启路由器，等待 1-2 分钟即可生效
⚠️ 不同品牌路径参考：华为（上网设置→IPv6）、小米（网络设置→IPv6）、TP-Link（高级设置→IPv6）、华硕（WAN→IPv6）
🔌 光猫（需配合路由器使用）
  • 1. 光猫一般由运营商配置，默认可能关闭 IPv6，先确认路由器已开启 IPv6 仍无法使用时再操作
  • 2. 访问光猫管理地址（常见：192.168.1.100，底部标签可查），输入运营商提供的账号密码（非路由器密码）
  • 3. 找到「IPv6 配置」→ 选择「桥接模式」或「启用 IPv6 转发」（不同运营商光猫设置差异大）
  • 4. 若不会操作，直接联系宽带运营商（电信/联通/移动），要求「开启 IPv6 服务」，运营商可远程配置
💡 关键提示：光猫需设置为「桥接模式」，路由器设置为「PPPoE 拨号」并开启 IPv6，或光猫为「路由模式」且已启用 IPv6，路由器自动获取即可
提示：若已启用 IPv6 仍无法访问，请检查路由器是否支持 IPv6 并已开启 | 检测网站：test-ipv6.com | 光猫配置可联系宽带运营商协助
